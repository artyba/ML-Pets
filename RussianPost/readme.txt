------
ПРОЕКТ
------
"Рекомендательная модель размещения постаматов во Владивостоке"

--------
ЗАКАЗЧИК
--------
Почта России (Хакатон "Цифровой прорыв")

------
ЗАДАЧА
------
Имеются данные:
- о жилых и нежилых строениях Владивостока(координаты, этажность, численность зарегистрированного населения по адресу, адрес) - 60898 строений
- данные "Points of Interest" о местах притяжения трафика(аптеки, банки, школы и т.д.) - 2478 точек)
Необходимо разработать модель, которая сможет предсказывать наиболее подходящие места во Владивостоке для размещения постаматов

-------
РЕШЕНИЕ
-------

1. Подготовка данных
++++++++++++++++++++
Данные предоставлены в формате .geojson. Датасеты преобразованы в читаемый формат, убраны неинформативные признаки.

2. Сбор дополнительных данных
+++++++++++++++++++++++++++++
Собраны данные:
- продуктовые магазины
- действующие постаматы конкурирующих компаний
- офисы компании СДЕК
(в ходе сбора решены задачи парсинга данных и пакетного геокодирования)

3. Генерация признаков
++++++++++++++++++++++
Для исследуемых Points of Interest сгенерированы признаки в радиусе 100/200/300/500 метров от каждой точки:
- численность населения в шаговой доступности
- количество коммерческих объектов в шаговой доступности
- статистики расстояний до коммерческих объектов для каждой точки - mean, std
- расстояние до ближайшего объекта каждого вида точек (расст. до ближайшего магазина, до ближайшего банкомата и т.д.)

4. Разработка модели и отбор признаков
++++++++++++++++++++++++++++++++++++++
Модель учится на данных о размещенных постаматах конкурентов.
Т.о. имеем задачу на частично размеченных данных.
известно, что 47 объектов принадлежат к классу 1, а из оставшихся 2431 любой объект может относиться к любому классу.

Часть датасета дополнительно размечена на уровне здравого смысла и предположений из которых строится модель.
На итоговом датасете построена модель, способная четко разграничить места, в которых постаматы уже размещены, и места, где их точно быть не может. 

На обученной модели произведен отбор признаков - 8 признаков, которые полностью сохраняют разделение классов при обучении упрощенной модели на них.

5. Прогнозирование и визуализация
+++++++++++++++++++++++++++++++++
Для датасета всех строений города сгенерированы признаки, которые были отобраны на этапе обучения - 8шт., и дан прогноз вероятности размещения постамата для любой точки города.
Построена карта вероятностей размещения постамата в каждой точке города.
Визуальная оценка результата по карте вероятностей показывает адекватность модели.
К сожалению, других данных для более объективной оценки у нас нет.

6. Масштабирование
++++++++++++++++++
Модель универсальна и может быть использована для любой местности, после дообучения на местых объектах.
Наиболее ресурсозатратным является генерация признкаков для большого количества объектов,
но данное действие является разовым для каждой территории и далее может происходить только уточнение и корректировка данных по мере необходимости.

В модель могут быть добавлены и учтены данные об экономической эффективности действующих постаматов Почты России, в результате чего будет произведена корректировка и даны намного более точные предсказания
